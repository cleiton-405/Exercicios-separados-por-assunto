Exercicio 5



Você vai criar um programa com menu interativo usando do...while, prompt e switch (ou if). O usuário pode escolher uma das opções para executar um exercício. O menu continua aparecendo até que o usuário escolha sair (opção 0).

======== MENU DE EXERCÍCIOS ======== 1 - Contar de 10 a 1 (decrescente) 2 - Verificar se número é primo 3 - Calcular média de 3 notas 4 - Contar vogais em uma palavra 5 - Gerar sequência de Fibonacci até N termos 0 - Sair ====================================
